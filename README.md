# Content Tracker

欢迎来到 Content Tracker —— 一个温柔记录你追过的故事、番剧、小说、电影与电视剧的工具。

> 心之所向，是梦的落脚处；沉默里藏着最真实的回响.

## 功能
- 导入和导出本地记录，轻松管理你的追踪清单
- 支持多种内容类型：事件、小说、番剧、电视剧、电影
- 简洁、直观、跨平台

## 安装与使用
1. 下载最新版本
2. 双击运行或在命令行执行
3. 导入已有记录或创建新的追踪列表
4. 轻松导出以备份或分享
